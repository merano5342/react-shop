# react-shop

這是一個使用 React 將 AC 購物車重新製作的專案

目前完成階段 -> 
v 使用 component 切分完版面
v 完成 ProgressControl, ProgressControl 與 Steps 的畫面串接

## 環境建置

1. 將專案下載至本地
打開 terminal 輸入：
```
git clone https://github.com/merano5342/react-shop.git
```

2. 下載所需套件
```
npm i
```

3. 開啟 localhost 在
localhost:3000/
```
npm start
```
